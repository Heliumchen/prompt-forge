# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.2] - 2025-08-26

### Added
- Better onboarding tutorial

## [1.0.1] - 2025-08-25

### Added
- TestSet sidebar optimization
- Support for Cmd+V import of standard Query data in TestSet

- Prompt Review functionality

- Refactoring issues

### Changed
- Better protection for local API keys

## [1.0.0] - Initial Release

### Added
- Core prompt engineering functionality
- Project and TestSet management
- Variable substitution system
- Batch testing capabilities
- Local-first data storage
- Version control for prompts
